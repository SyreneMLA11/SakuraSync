# SakuraSync
**🌸 Sakura Sync**

*Automated timezone intelligence for students living across two clocks.*

Sakura Sync turns any webpage into a calendar entry. Open it on a WhatsApp message, an event flyer, or a screenshot from a group chat, and it reads the details for you, pulling out the title, time, and location without manual entry. No more converting WIB to PST by hand before a 9am call that falls at midnight back home.

The extension uses an AI powered parser built to handle the inconsistent way event details actually appear online, with text highlighting and screenshot upload available as fallback inputs when a page is too unstructured to read automatically. Every result passes through a review step before it reaches your calendar, so you stay in control of what gets created. Time zones convert instantly and can be set to any pairing beyond the WIB to PST default, events are color coded by category, and a three stage reminder system covers one day before, the morning of, and ten minutes before, so nothing is missed across time zones.

Lightweight, precise, and purpose built for international students balancing academic and personal schedules across borders.
